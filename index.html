GVM the Groovy enVironment Manager

GVM is a tool for managing parallel Versions of multiple Software Development Kits on most Unix based systems. It provides a convenient command line interface for installing, switching, removing and listing Candidates.

Candidates that will initially be supported are:

  • Groovy
  • Grails
  • Griffon
  • Gradle

This list will grow beyond Groovy candidates.

Supported platforms are:

  • Linux
  • Mac OSX
  • Cygwin
  • Solaris
  • FreeBSD (with bash installed)

GVM only depends on bash, curl, zip and unzip to be present on your system. The installer will check for these before completing the installation process. It will then create a .gvm folder in your home directory, and neatly install all candidates beneath it.

GVM also has a server-side component written with vertx enabling it's dynamic behavior. New available candidates and versions can be added to GVM without the need for local re-installation. It also facilitates easy installation and selfupdate of the client-side components as well as a facility for live notifications.

The server uses MongoDB for persisting candidates and gathering of usage statistics.

Installation

Open your favourite bash terminal and enter the following:

curl -s beta.gvmtool.net | bash

If the environment needs tweaking for GVM to be installed, the installer will prompt you accordingly and ask you to restart.

Usage

After installation, you will need to run the code snippet that the installer prompts you to. Alternatively you could open a new terminal. Now run the following:

Groovy enVironment Manager 0.4
GVM server 0.4 on vertx 1.2.3.final

=== BROADCAST ==========================================
Some live message...
========================================================

Usage: gvm <command> <candidate> [version]

   command   : install, uninstall, list, use, current, selfupdate or help
   candidate : grails,groovy
   version   : optional, defaults to latest stable if not provided

eg: gvm install grails 2.1.0

To install the current default grails version, simply type:

gvm install grails

and then:

gvm use grails

That's it! As easy as that!
